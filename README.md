변경 예정 사항
1. 전체적인 구조 변경
   1. 각 객체별로 수많은 업데이트문이 있음 -> GameManager(아두이노) 중심으로 변경
   2. 해상도 고정
   3. 설정창 따로 생성

2. 크기 변경
   1. 좌표 1당 1cm -> 1m
   2. 기대 효과: 중력을 변경하지 않아도 실제와 비슷한 속도로 굴러감

3. 거리 측정 로직 변경
   1. 센서와 공의 3차원 공간 상에서의 거리 사용 -> 센서에서 레이케이스 발사하여 거리 측정
